Dots & Boxes Game


This is a classic childhood game. Be the last person to complete the box to get a point. The person with the most points wins the game. I created this app with Javascript and Object Oriented Programming. There is an option for 2 players or for 1 player which useses an AI.

Random Meal Generator


The purpose of the Random Meal Generator is for those who just can't figure out what to cook. On the home page there is the option to create a meal. The user inputs a type of food (enchiladas, soup, cake) and an ingredient, and then a random meal is shown using the Recipe Puppy API. You also have the option to register, which will then take you to your own personal page. On your personal page you will be able to plan a weeks worth of meals using the same format.

HTML/CSS/Javascript/Node/Mongo/Express/Angular

Github Heroku

Bananas & Butterflies


This is a translator app. The user can enter words and the language they would like it translated into. They can also take a quiz in any language they choose. The user can see their stats after the quiz has finished.
